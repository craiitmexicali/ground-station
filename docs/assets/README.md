# Assets Directory

Este directorio contiene los recursos gráficos del proyecto:

## Capturas de Pantalla

Añade aquí las capturas de pantalla del proyecto:

- `dashboard.png` - Captura del dashboard principal
- `dashboard-preview.png` - Preview para el README
- `logs.png` - Captura de la consola de logs

## Cómo generar capturas

1. Ejecuta la aplicación con `npm start`
2. Haz clic en "Iniciar Simulación"
3. Espera unos segundos para que se generen datos
4. Usa la herramienta de captura de tu sistema operativo

## Tamaños recomendados

- Preview para README: 1200x675 px (16:9)
- Capturas de sección: 800x450 px
